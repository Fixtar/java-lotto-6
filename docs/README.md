# 기능 목록

## 입력 받기
- [ ] 로또 구입 금액 입력받기
- [ ] 당첨 번호 입력받기
- [ ] 보너스 번호 입력받기

## 로또 게임 애플리케이션
- [ ]  
- [ ] ㅁㄹㅇ

## 로또 번호 생성기
- [ ] 1 ~ 45 까지의 숫자중 중복되지 않는 수 6개 뽑기 (로또 발행)

## 로또 결과 판별기
- [ ] 당첨 번호 6개와 보너스번호, 로또 객체의 정보를 통해 몇등인지 판별하기
- [ ] 당첨 번호 6개로 판별하기
- [ ] 보너스 번호 판별하기

## 로또 객체
- [ ] 숫자가 6개인지 검증하기
- [ ] 당첨 번호 개수 반환하기

## 결과 출력기
- [ ] 발행한 로또 번호 출력하기
- [ ] 당첨 내역 출력하기
- [ ] 수익률 출력하기
